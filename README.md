# lockey33-agent-skills

Skills for Diego, Laurent's AI agent. Each subfolder contains a `SKILL.md` describing a capability.

## Available skills

- `session-handoff` — resume a Claude/Codex/Kimi session from a session ID
- `telegram-output` — formatting rules for Telegram responses
- `memory` — long-term memory management across sessions
- `mstr-btc` — dashboard MSTR/BTC : régime BTC, mNAV, recommandation d'asset, kill switches
- `sonocast-tiktok` — générer des vidéos TikTok à partir d'épisodes de podcast via Sonocast
- `lifeos-standup` — standup du matin : priorités, métriques, leads, candidatures
- `lifeos-rapport` — bilan de fin de journée : accompli, métriques, actions du lendemain
- `lifeos-tasks` — gestion des tâches Notion : priorités, stagnations, challenge objectifs
- `lifeos-x` — contenu X : tendances, génération brouillons, publication (approbation requise)
- `lifeos-leads` — leads LinkedIn et candidatures emploi
